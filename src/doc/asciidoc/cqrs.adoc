= CQRS mit Axon

Command Query Responsibility Segregation in Theorie +
Und die Praxis mit dem Axon Framework

== Bauen wir einen Customer Service!

=== Was soll er alles können?
[%step]
* Registrierung eines Kunden
* Abrufen von Kundendaten
* Änderung von Kundendaten
* Löschen von Kunden

=== Good old - CRUD

[source,javascript]
----
include::{sourcedir}/resources/customer.json[indent=0]
----
